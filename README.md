## Getting Started

### Installation

1. **Clone the Repository:**

   ```bash
   git clone https://github.com/LucaDiba/monyfox.git
   ```

2. **Navigate to the Project Directory:**

   ```bash
   cd monyfox
   ```

3. **Install Dependencies:**

   ```bash
   yarn
   ```

4. **Run the Application:**
   ```bash
   cd apps/client/dashboard
   yarn dev
   ```
